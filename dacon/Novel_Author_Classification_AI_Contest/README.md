## 사용한 방법 & 알고리즘

- DNN 모델과 CNN 모델을 만들어 학습 후 앙상블
- 텍스트 전처리
  - DNN은 SentencePieceTrainer를 이용하여 전처리 하였다.
  - CNN은 Bert를 이용하여 전처리를 하였다.
- K-Fold를 사용하여 성능이 제일 좋은 모델을 앙상블에 사용하였다.
- DNN과 CNN모델 3개를 사용하였다.
